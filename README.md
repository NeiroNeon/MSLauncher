# Bedrock Control Platform — Stage 0 Foundation

Android-first cross-platform Minecraft Bedrock management platform.

## Supported Platforms

- Android (Termux)
- Linux
- Windows

## Current Stage

Stage 0 — Foundation

## Features

- pnpm monorepo
- TypeScript setup
- shared tooling
- Android-first scripts
- build pipeline skeleton
- runtime bootstrap
- project structure

## Quick Start

### Android / Termux

```bash
chmod +x scripts/android/setup.sh
./scripts/android/setup.sh
```

### Development

```bash
pnpm install
pnpm dev
```
